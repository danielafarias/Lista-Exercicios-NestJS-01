## 🚀 Instalando `<SortArray-NestJS>`

Para instalar o `<SortArray-NestJS>`, siga estas etapas:

Clone o projeto:
```
<git clone https://github.com/danielafarias/SortArray-NestJS.git>
```
Em seguida, adicione as bibliotecas e dependências necessárias:
```
<cd SortArray-NestJS>
```
```
<cd server>
```
```
<npm init>
```
